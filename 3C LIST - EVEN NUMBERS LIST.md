## LIST - REVERSING NUMBERS

---

### AIM  
To write a Python program to reverse the members of a given list.

---

### ALGORITHM

1. Begin the program.  
2. Get the list
3. Use reverse function to reverse the list
4. print the list
5. Terminate the program
---

### PROGRAM

```
l=[14,15,6,7,9,2,4]
l.reverse()
print(l)
```
